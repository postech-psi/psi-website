                    <h2>Avionics Flight Computer</h2>
                    <p class="project-meta">● Active Development</p>
                    <p>A high-performance, RTOS-based avionics flight computer for the Arduino Portenta H7, designed for autonomous rocket recovery and telemetry. The system integrates multi-sensor fusion—combining GNSS, IMU, and BMP3XX barometric data with an Unscented Kalman Filter (UKF)—to provide precise real-time state estimation and robust flight stage detection.</p>
                    <div class="project-features">
                        <h3>Key Features</h3>
                        <ul>
                            <li>Multi-sensor fusion (GNSS, IMU, Barometer)</li>
                            <li>Unscented Kalman Filter for state estimation</li>
                            <li>RTOS-based concurrent processing</li>
                            <li>Autonomous flight stage detection</li>
                            <li>XBee telemetry transmission</li>
                            <li>Reliable parachute deployment logic</li>
                        </ul>
                    </div>
